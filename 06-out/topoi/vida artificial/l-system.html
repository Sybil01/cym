L-System Controls Guide

This visualization generates trees using an L-System, a set of rules that creates complex patterns from a simple string.

How to Use:

  • Sliders: Adjust the tree's growth speed, branch density, and audio volume.
  • Rule Input: Type your own L-System rule to create custom trees.
  • Presets: Select from a list of predefined patterns to get started.
  • Camera: Click and drag to rotate. Scroll to zoom.

L-System Pattern Symbols:

  • F : Draw a branch forward.
  • + : Turn right by a set angle.
  • - : Turn left by a set angle.
  • [ : Push the current position and angle onto a stack (save state).
  • ] : Pop the last position and angle from the stack (return to saved state).
